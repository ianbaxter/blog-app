# Blog

The front end for a simple blogging web application.

## Setup

Clone the project: 
```
git clone https://github.com/ianbaxter/blog-app
```


Navigate to the project directory:
```
cd blog-app
```

Install dependencies:
```
npm install
```

Start the site with:
```
npm start
```
